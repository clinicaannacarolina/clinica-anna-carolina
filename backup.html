ℹ️ Sobre o backup
O backup completo exporta todos os dados em formato JSON — pode ser guardado como arquivo de segurança.
Os arquivos CSV são ideais para abrir no Excel.
Recomendado: fazer backup ao menos 1x por semana e guardar em local seguro (Google Drive, etc).
